# Interface: DataMap

This map registers the type of the `data` key of a `VFile`.

This type can be augmented to register custom `data` types.

## Example

```ts
declare module 'vfile' {
  interface DataMap {
    // `file.data.name` is typed as `string`
    name: string
  }
}
```
